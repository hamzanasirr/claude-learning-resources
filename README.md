## 🟠 Tier 3 — Prompt Engineering (Universal Skill — Everyone Should Do This)

This is where non-technical and technical audiences converge. Prompting is the single highest-leverage skill in the AI era.

### 7. Anthropic's Interactive Prompt Engineering Tutorial

Two formats — pick the one that suits you:

| Format | Link | Best For |
|---|---|---|
| Jupyter Notebook | https://github.com/anthropics/prompt-eng-interactive-tutorial | Developers |

**What's covered across 9 chapters:**
- Basic prompt structure
- Role assignment
- Chain-of-thought reasoning
- Multi-shot examples
- XML tags for structure

**The 5 highest-impact prompting techniques:**
1. Use XML tags to structure your prompts
2. Ask for evidence or reasoning before a conclusion
3. Assign a specific role to Claude
4. Request step-by-step thinking
5. Show examples instead of writing long instructions

## 🔴 Tier 5 — Advanced Developer / Agentic AI

For those going deep into automation, agents, and production systems.

### 11. Introduction to Model Context Protocol (MCP)
🔗 https://anthropic.skilljar.com/introduction-to-model-context-protocol

Build MCP servers and clients from scratch in Python. Covers the three core primitives — tools, resources, and prompts — to connect Claude with external services.

### 12. Introduction to Agent Skills
🔗 https://anthropic.skilljar.com/introduction-to-agent-skills

Build, configure, and share reusable Skills in Claude Code — markdown instructions that Claude automatically applies to the right tasks at the right time.

### 13. Introduction to Subagents
🔗 https://anthropic.skilljar.com/introduction-to-subagents

Create sub-agents in Claude Code to manage context, delegate tasks, and build specialized workflows that keep your main conversation clean and focused.

### 14. Claude Code in Action
🔗 https://anthropic.skilljar.com/claude-code-in-action

A 21-lesson course covering Claude Code from setup through GitHub integration and hooks — for developers who want AI assistance that works as a true coding partner.

### 15. Model Context Protocol: Advanced Topics
🔗 https://anthropic.skilljar.com/model-context-protocol-advanced-topics

Production-grade MCP patterns — sampling, notifications, file system access, and transport mechanisms for engineers already comfortable with MCP fundamentals.
